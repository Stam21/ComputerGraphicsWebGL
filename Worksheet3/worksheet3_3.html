Reflect on the theory of affine transformations and viewing transformations by doing the following:
List the transformation matrices that you used in Parts 1 and 2 (use general expressions/functions rather than concrete numbers).
For each cube, write down a formula showing how the matrices were concatenated to become the current transformation matrix (CTM) that was used to transform the vertices in the vertex shader.

Answer:

Part 1:
The matrices used were the result of the lookAt function and the orthographic projection. Therefore,the CTM formula used to transform the vertices in the vertex shader is the following :
CTM = lookAt x ortho
This way the camera Position and orientation is altered to show the unit cube in isometric view.

Part 2:
For all three cubes a new matrix was introduced, the perspective matrix. Also, in order to fit in the same canvas all three cubes, a translation matrix was used as well. The three-point perspective illustrated on the left of the canvas uses the following formula:
CTM = ortho x perspective x lookAt x translate
The two-point perspective illustrated on the right of the canvas uses the following formula:
CTM = ortho x perspective x lookAt x translate
The one-point perspective illustrated on the middle of the canvas uses the following formula:
CTM = ortho x perspective x lookAt
