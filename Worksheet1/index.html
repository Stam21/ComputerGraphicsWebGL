The purpose of this set of exercises is to get started with WebGL (if you prefer, you can use WebGPU instead of WebGL). You will setup a WebGL application from scratch, create a canvas and a WebGL context, compile and use simplistic shader programs, setup the needed buffers for drawing, and draw and animate simple shapes.
